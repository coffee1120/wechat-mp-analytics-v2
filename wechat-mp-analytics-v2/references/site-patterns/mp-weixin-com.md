## 平台特征

- **登录方式**: 微信扫码（无法绕过）
- **会话保持**: 约 2 小时，超时需重新扫码
- **反爬级别**: 中等（CDP 模式可绕过）
- **页面加载**: 动态渲染，需等待 JS 执行
- **数据更新**: 实时，但昨日数据在次日 0 点后更新
- **投流资格**: 新号需满足一定条件（粉丝数、原创声明等）

## 有效模式

### URL 模式（2026-03 验证）

- **首页**: `https://mp.weixin.qq.com/cgi-bin/home?t=home/index&lang=zh_CN&token={token}`
- **内容助推**: `https://mp.weixin.qq.com/cgi-bin/mptrafficboost?action=dashboard&token={token}`
- **全部激励**: `https://mp.weixin.qq.com/cgi-bin/mptrafficboost?action=incentive&token={token}`
- **数据分析**: `https://mp.weixin.qq.com/cgi-bin/rpt?token={token}`

### Token 获取

- token 在登录后的 URL 参数中
- token 有效期约 2 小时
- 失效后需重新登录获取
- **不要手动构造 token**，从当前会话提取

### 操作策略

**最佳路径**:
1. 直接导航 URL（不点击菜单）
2. 等待 3 秒让动态数据加载
3. 截图确认页面结构
4. 执行 JS 提取数据
5. 关闭自创 Tab

**URL 提取方法**:
- 从 DOM 提取完整链接（保留所有参数）
- 不要手动构造 URL（可能缺失必要参数）
- 示例：`document.querySelector('a[href*="mptrafficboost"]').href`

## 已知陷阱

### 1. 登录态过期 ⭐⭐⭐

**现象**: 页面跳转到 `https://mp.weixin.qq.com/cgi-bin/bizlogin?action=startlogin`

**原因**: 
- token 过期（>2 小时）
- Cookie 失效
- 长时间无操作

**解决**: 
1. 截图登录页
2. 告知用户："请微信扫码登录公众号后台"
3. 等待用户确认"已登录"
4. 刷新页面或重新导航

**不要**: 尝试自动登录（安全风险）

### 2. 数据延迟 ⭐⭐

**现象**: 昨日数据显示为 0 或与预期不符

**原因**: 
- 平台在次日 2:00-6:00 间更新数据
- 月初/周一可能延迟更久

**解决**: 
- 提示用户："数据可能未完全更新，建议 8:00 后查询"
- 或标注"数据更新时间：MM-DD HH:MM"

### 3. 选择器失效 ⭐⭐⭐

**现象**: 提取到的数据为空或 NaN

**原因**: 
- 公众号后台改版（常见于季度更新）
- A/B 测试导致页面结构不同

**解决**: 
1. 截图确认页面结构
2. 用 `/eval` 探索 DOM：`document.body.innerHTML.substring(0, 2000)`
3. 更新选择器
4. 更新站点经验文件（标注日期）

### 4. 投流页面 404 ⭐

**现象**: 访问 `/cgi-bin/mptrafficboost` 返回 404 或错误页

**原因**: 
- 新号无投流资格（需满足粉丝数、原创等条件）
- token 过期
- 功能入口调整

**解决**: 
1. 检查账号资格（查看是否有"内容助推"菜单）
2. 刷新 token（重新登录）
3. 从首页点击"内容助推"进入（不直接导航）

### 5. 并发限制 ⭐

**现象**: 部分请求失败或返回空数据

**原因**: 
- 短时间内打开过多 Tab（>5 个）
- 请求频率过高

**解决**: 
- 限制并发 Tab 数量（建议≤3）
- Tab 间延迟 1-2 秒
- 用子 Agent 分治时注意控制并发

### 6. "全部激励"页面难访问 ⭐⭐

**现象**: 点击"全部激励"无反应或页面不存在

**原因**: 
- 该页面是动态加载的模态框/下拉层
- URL 不直接暴露

**解决**: 
1. 在内容助推页面点击"全部激励"文本/按钮
2. 等待模态框加载
3. 截图提取数据
4. 不要尝试直接导航
